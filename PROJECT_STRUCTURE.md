## ğŸ› Common Issues & Solutions

### Issue: Theme not persisting

**Solution**: Check `client/global.css` and `tailwind.config.ts`
- CSS variables format: `--name: hue saturation% lightness%;`
- Tailwind theme: `color: "hsl(var(--name))"`

### Issue: Language not updating

**Solution**: Ensure `LanguageProvider` wraps the app in `App.tsx`
- Check `useLanguage()` hook is imported correctly
- Verify `client/lib/i18n.ts` has translations for all keys

### Issue: Tool not showing

**Solution**: Check three places:
1. Tool component exists: `client/components/tools/[Tool].tsx`
2. Imported in `client/pages/Index.tsx`
3. Added to `tools` array with all required properties

### Issue: Styling looks wrong

**Solution**: Check CSS variable format:
- âŒ Wrong: `--color: rgb(100, 200, 300)`
- âœ… Right: `--color: 200 100% 50%`
